**Root cause sebelumnya:**
- `clamp(14px, 1.5vw, 20px)` — di HP 390px lebar, `1.5vw` = 5.8px, jadi `clamp` ambil minimum 14px tapi `height: clamp(48px, 6vw, 78px)` di HP = 23px → input **gepeng**

**Solusi — media query eksplisit per breakpoint:**

| Breakpoint | Target |
|---|---|
| Default | Desktop — padding 64px, font 36px, input 56px |
| `≤ 768px` | Tablet mini (iPad Mini, Tab A) — padding 40px, font 26px, input 50px |
| `≤ 480px` | HP Android & iPhone — padding 24px, font 22px, input 46px, tombol stack vertikal |
| `≤ 360px` | iPhone SE / Android kecil — padding 20px, input 44px |
